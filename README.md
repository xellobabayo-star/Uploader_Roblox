## рҹҺ« Sistem Tier & Invite

```
TRIAL      вҶ’   3 upload / bulan   (default semua user baru)
BEGINNER   вҶ’  50 upload / bulan   (redeem kode invite)
PRO        вҶ’   вҲһ unlimited         (redeem kode invite)
```

**Cara buat kode invite:**
1. Login ke `/admin`
2. Menu **Invite Codes** вҶ’ **BUAT KODE**
3. Pilih tier, max uses, expiry date
4. Share kode ke user

**User redeem:**
- Saat register вҶ’ langsung dapat tier
- Di dashboard вҶ’ Settings вҶ’ Invite Code

## рҹҺ® Cara Pakai di Roblox

Setelah upload dengan speed **XГ—**, set di script Roblox:

```lua
-- Rumus: PlaybackSpeed = 1 Г· Upload Speed
-- Contoh upload 4Г— вҶ’ PlaybackSpeed = 0.25

local Sound = workspace.MySound
Sound.SoundId = "rbxassetid://ASSET_ID_DISINI"
Sound.PlaybackSpeed = 0.25   -- sesuaikan dengan speed yang dipakai
Sound.Looped = true
Sound:Play()
```

| Upload Speed | PlaybackSpeed | Cocok untuk |
|---|---|---|
| 2Г— | `0.5` | Lagu вүӨ 14 menit |
| 4Г— | `0.25` | Lagu вүӨ 28 menit |
| 8Г— | `0.125` | Lagu вүӨ 56 menit |
| 10Г— | `0.1` | Lagu ~1 jam |

## вҡ пёҸ Catatan Penting

> **FFmpeg** вҖ” Fitur speed & pitch butuh FFmpeg. Di Railway/VPS otomatis tersedia via `@ffmpeg-installer/ffmpeg`. Di Vercel tidak tersedia.

> **Database** вҖ” Data di `data/xello.sqlite`. Jangan hapus folder ini! Backup berkala disarankan.

> **File Size** вҖ” Max upload 150MB per file. Cukup untuk audio panjang.

> **Vercel** вҖ” Bisa deploy tapi fitur YouTube download & SSE streaming terbatas. Disarankan Railway untuk produksi.
